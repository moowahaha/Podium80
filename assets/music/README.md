# assets/music — background music (paired with backgrounds)

Drop a track here and it loops automatically on the matching screen/event — no code change. The key
matches the paired backdrop, so music and background go together. **`.ogg` recommended** (small,
loops cleanly); `.mp3` / `.wav` also work.

| File | Plays on | Paired background |
|------|----------|-------------------|
| `menu.ogg` | Title, Country Select, Hub, Results | `assets/menu/background.png` |
| `track.ogg` | Sprint + Hurdles | `assets/stadium/track.png` |
| `long_jump.ogg` | Long Jump | `assets/stadium/long_jump.png` |
| `hammer.ogg` | Hammer Throw | `assets/stadium/hammer.png` |
| `swim.ogg` | 100m Swim | `assets/stadium/swim.png` |
| `podium.ogg` | Podium ceremony | `assets/stadium/podium.png` |

The same track keeps playing seamlessly across screens that share a key (e.g. all the menu screens),
so it won't restart on every navigation.
